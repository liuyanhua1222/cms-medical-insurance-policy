# CMS Medical Insurance Policy Skill

**OpenClaw AI Agent Skill** - 查询各地异地就医备案政策，提供门诊、住院、报销额度等关键信息。

## 核心定位

这是一个 **AI提示词驱动的Skill**，用于让 AI Agent 能够：
- 理解用户关于异地就医政策的查询
- 使用提示词模板生成结构化政策解读
- 以自然语言直接回答用户问题
- 提供官方查询渠道作为参考

---

## 快速开始

### Agent 使用方式

用户询问异地就医政策时，AI直接使用提示词模板生成回答：

**示例对话**：

```
用户：北京异地就医备案政策
AI：使用提示词模板生成完整政策解读
```

### 提示词模板

详见 [SKILL.md](SKILL.md) 中的完整提示词模板。

---

## 核心功能

### 1. 地区政策查询

根据用户提供的地区，使用提示词模板生成该地区的异地就医备案政策解读。

### 2. 专题咨询

- 门诊报销政策
- 住院报销政策
- 备案流程指导
- 报销额度说明

### 3. 特殊人群政策

- 退休人员
- 在职职工
- 急诊患者
- 慢性病患者

### 4. 官方渠道指引

当无法确定具体政策时，提供官方查询渠道：
- 国家医保服务平台APP
- 12333服务热线
- 地方医保局官网

---

## 三优先级回答策略

### 1️⃣ 首选：直接生成政策解读

使用内置的提示词模板，根据用户地区和问题，直接生成结构化回答。

### 2️⃣ 次选：提供通用政策

如无法确定具体地区，提供全国通用政策和平均比例。

### 3️⃣ 最终：指引官方查询

提供官方查询渠道，让用户自行核实最新政策。

## 回答格式示例

```markdown
## 📋 北京市异地就医备案政策

### 基本信息
- 适用地区：北京市
- 政策来源：国家医保局、北京市医保局

### 门诊使用
- 是否可用：✅ 是
- 报销比例：55%-70%（根据医院级别）
- 注意事项：需提前备案

### 住院使用
- 是否可用：✅ 是
- 报销比例：70%-85%（根据医院级别）

### 备案流程
- **线上**：国家医保服务平台APP、微信/支付宝小程序
- **线下**：各区医保经办机构

### 注意事项
> 具体政策以当地医保部门最新公布为准
```

## 常见问题

### Q: 这个Skill需要安装什么依赖吗？
A: 不需要。这是纯提示词驱动的Skill，不需要Python脚本或爬虫。

### Q: AI回答的政策信息准确吗？
A: AI基于通用医保政策和提示词模板生成回答，仅供参考。具体政策以当地医保部门公布为准。

### Q: 如何获取最新的地区政策？
A: 建议通过国家医保服务平台APP、12333热线或当地医保局官网查询。
